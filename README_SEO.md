# SEO y publicación - Rancho Quilla

## Antes de publicar

1. Reemplazar en `index.html` todos los enlaces `https://wa.me/5491112345678` por el WhatsApp real de reservas.
2. Cuando exista dominio, agregar en `<head>`:
   - `<link rel="canonical" href="https://tudominio.com/">`
   - `<meta property="og:url" content="https://tudominio.com/">`
   - Usar URL absoluta para `og:image`.
3. Renombrar `sitemap.example.xml` a `sitemap.xml`, actualizar el dominio y descomentar la línea Sitemap de `robots.txt`.
4. Configurar Google Search Console y subir el sitemap.
5. Crear o completar Perfil de Empresa en Google con el mismo nombre: Rancho Quilla.

## Mejoras incluidas

- Título SEO y meta description orientados a búsqueda local.
- H1 único con keyword principal.
- Copy natural con variantes de búsqueda.
- Fotos reales optimizadas en WebP.
- Open Graph para compartir en WhatsApp/redes.
- Schema `LodgingBusiness` y `FAQPage`.
- FAQ visible con preguntas frecuentes para SEO long-tail.
- Botones de WhatsApp con mensaje precargado.
- Mapa embebido del Dique Cabra Corral.
